# Contribuindo para o PortfolioHUB

Fico feliz com o seu interesse em contribuir! Siga os passos abaixo:

1. Faça um Fork deste repositório.
2. Crie uma nova branch com a sua feature: `git checkout -b minha-feature`.
3. Faça as suas alterações e confirme: `git commit -m "feat: adiciona nova funcionalidade"`.
4. Envie para o GitHub: `git push origin minha-feature`.
5. Abra um Pull Request para a branch master detalhando as mudanças.

Toda contribuição será revisada e testada antes de ser aceita no ambiente de produção.
